# Study Café Optimizer

## Overview

Study Café Optimizer is a Java Swing desktop application developed for the CCS4202 Design and Analysis of Algorithms course.

The system helps students select the best combination of study cafés within a given budget by comparing three optimization algorithms.

## Algorithms Implemented

- Dynamic Programming
- Brute Force
- Greedy Algorithm

## Features

- Budget-based café optimization
- Dynamic Programming process visualization
- Brute Force process visualization
- Greedy algorithm visualization
- Algorithm comparison
- Interactive Java Swing GUI

## How to Run

1. Open the project in Visual Studio Code.
2. Install the Java Extension Pack.
3. Open `Main.java`.
4. Click **Run**.

Requirement: Java 25 and Visual Studio Code with the Java Extension Pack installed.
